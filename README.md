IaaS:<br/>
IaaS ist ein Service, bei dem eine ganze Infrastruktur zur Verf端gung gestellt wird. Beispielsweise AWS - Services, wo man mehrere VM's in einem Netzwerk erstellen kann.<br/>

PaaS:<br/>
Hier ist der Service, wo man eine Plattform anbietet, beispielsweise ein Webserver, oder einfach eine limitierte Umgebung. Es liegt unter IaaS. Das heisst, alles was PaaS ist, ist gleichzeitig auch IaaS.<br/>
https://github.com/ristifilip/ristiflip/blob/main/PaaS <br/>

SaaS:<br/>
Hier stellt man eine Software zur Verf端gung, bzw. als Dienstleistung. Beispielsweise stellt Abraxas die Software Abacus zur Verf端gung. Dies liegt unter PaaS.<br/>
https://github.com/ristifilip/ristiflip/blob/main/SaaS <br/>
